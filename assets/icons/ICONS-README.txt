Icon Requirements for AI Prompt Enhancer Extension
===================================================

You need to create three PNG icon files with the following specifications:

icon-16.png
-----------
- Dimensions: 16x16 pixels
- Format: PNG with transparency
- Usage: Browser toolbar icon (small size)
- Design: Simple, recognizable version of the logo

icon-48.png
-----------
- Dimensions: 48x48 pixels
- Format: PNG with transparency
- Usage: Extension management page
- Design: Medium detail version of the logo

icon-128.png
------------
- Dimensions: 128x128 pixels
- Format: PNG with transparency
- Usage: Chrome Web Store listing, high-res displays
- Design: Full detail version of the logo
